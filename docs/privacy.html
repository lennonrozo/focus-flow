Privacy Policy for FocusFlow

Last Updated: January 22, 2026

Overview

FocusFlow is committed to protecting your privacy. This extension is designed specifically for cancer survivors with aphasia and hemineglect to improve web accessibility. We respect your data and operate with transparency.

What Data We Collect

We do NOT collect, store, or share any personal information about you. FocusFlow respects your privacy entirely.

How the Extension Works

Local Features (No Data Sent)

  • Visual Shift: Repositions page content to the right side. Runs entirely in your browser.
  • Highlighter Ruler: Spotlight effect following your mouse. No data collection.
  • Flashing Anchor Bar: Visual aid on the left edge. Runs locally in your browser.
  • User Preferences: Stored locally in your browser using Chrome storage. Never sent anywhere.

Text Simplifier (Backend Processing)

When you use the AI text simplifier feature:

  • Text you select is sent to our backend API for processing by Google's Gemini AI
  • We do NOT store or log the text you send
  • We do NOT share your text with third parties (except Google Gemini API, as necessary for simplification)
  • Simplified results are cached locally in your browser for performance and to avoid redundant API calls
  • You can clear the cache at any time through your browser settings

Permissions Explained

Access your data for all websites (<all_urls>)

  • Required to apply visual accessibility features (visual shift, highlighter, anchor bar) to any website you visit
  • Used to extract and simplify text you select
  • No data is stored or transmitted based on this permission

Inject scripts into pages (scripting)

  • Required to apply the visual features (shift, highlighter, anchor bar) to the webpage content

Storage (storage)

  • Stores your extension preferences (which features are enabled, visual shift amount, etc.)
  • Stores cached simplified text to improve performance and reduce API calls
  • All stored data remains on your device

Third-Party Services

Google Gemini API

  • FocusFlow uses Google's Gemini API for text simplification
  • When you use the text simplifier, your text is sent to Google's servers
  • Google's privacy policy applies: Google AI Privacy
  • We recommend reviewing Google's policy if you have concerns about API processing

Data Security

  • All local data is encrypted by the Chrome browser's secure storage
  • Communication with the backend API uses HTTPS encryption
  • We do not store usernames, passwords, or payment information

Your Rights

  • Access: You can review all cached data in Chrome's application storage (DevTools → Application → Storage)
  • Delete: You can clear the extension's cache and preferences at any time through Chrome Settings
  • Opt-out: You can disable features or uninstall the extension at any time

Changes to This Policy

We may update this privacy policy occasionally. The latest version will always be available at this URL. Continued use of the extension means you accept any changes.
